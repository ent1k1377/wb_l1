1. Я правильно понимаю, что передача в функцию/метод всегда происходит по значению?
То есть при передаче по значению копируется объект в функцию/метод, а при передаче по указателю копируется указатель
в котором храниться ссылка на объект, что и позволяет работать с оригинальным объектом и не приходиться использовать лишнюю память.
2. Как правильно использовать ресивер? Ресивер по значению или по указателю. Я прочитал статью и там было
много ситуаций когда лучше по значению и когда лучше по указателю, а еще все было сказано, что
`
Общее правило для обоих типов
Если ваша структура имеет метод Value Receiver, то все остальные методы этой структуры должны быть Value Receivers.
То же самое для указателей.
Если у вашей структуры есть метод Pointer Receiver, все остальные методы этой структуры должны быть Pointer Receivers.
`
